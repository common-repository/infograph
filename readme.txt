== Description ==

Display your blog info with graphs. Use [velocity] , [mycategories] , [bymonth]  and [mypopularposts] shortcodes for displaying different charts. Just try them.

== Installation ==

Simple:

1. Descompress the file
2. Upload the directory `infograph/` to the `wp-content/plugins/` directory.
3. Activate the plugin
4. Use [velocity] , [mycategories] , [bymonth]  and [mypopularposts] shortcodes for displaying different charts;
